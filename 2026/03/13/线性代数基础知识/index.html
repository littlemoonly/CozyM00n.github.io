线性空间 / 向量空间

向量空间是全部n维向量 连同 加法和数乘运算，合称为向量空间。

向量空间$\bold{R^3}$ 包含所有3维向量 / 所有具有n个分量的列向量。

A是mxn矩阵，A的列空间是Rm的子空间，即所有列可能的组合，即Ax

矩阵的秩

矩阵 A 的真正大小是由秩(rank)决定的。

A的秩是主元的数量。

秩1矩阵

当所有的行都是主元行的倍数时，秩r = 1；

列空间 C(A)

对于m*n矩阵：

列空间C(A) 是 m维向量空间，是A 的所有列向量生成的子空间；

设 A = [a₁ a₂ … aₙ] （m×n矩阵） 则：C(A) = span{a₁, a₂, …, aₙ} = {c₁a₁ + c₂a₂ + … + cₙaₙ : c₁,c₂,…,cₙ ∈ ℝ}；

列空间维数 dim(C(A)) = rank(A) = 列向量中线性无关向量的最大个数

零空间 N(A) NullSpace

对于Am*n（m个方程，n个未知数）

N(A) = {x ∈ ℝⁿ : Ax = 0}，A的零空间是n维向量空间；

如何理解？

零空间包含所有被矩阵A”消灭”的向量，即经过线性变换A后变成零向量的所有向量。

零空间维数 = n - r = 自由变量的个数 = 系统的”自由度”

左零空间 N(A^T) Left NullSpace

是矩阵AT 的零空间；

N(AT) = y ∈ ℝᵐ : ATy = 0；

左零空间包含所有与列空间正交的向量；

如果 y ∈ N(A^T)，那么 A^T y = 0，这意味着 y 与 A 的每一列都垂直；

线性变换的概念

线性变换 T 使得每个向量 v 变成 T(v)

判断是否为线性变换的例子

  1. 平移不满足线性

T(v + w) + u0 = T(v) + u0 + T(w) + u0 != T(v) + T(w) + u0

线性变换 + 平移变换 = 仿射变换

  1. 旋转满足线性

基底如何变换决定了整个空间如何变换

线性变换-求导 T = d / dx

在选择输入与输出基底之后，线性变换的矩阵随之确定

求导变换 T 有一个核(即 u = 常数，u(x) = C )，对应的矩阵A有一维零空间。
