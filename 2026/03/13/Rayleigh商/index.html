逐个观察奇异值向量

目标 : 最大化 λ : $$ R(x)=\frac{\|A\boldsymbol{x}\|^2}{\|\boldsymbol{x}\|^2}=\frac{\boldsymbol{x}^{\mathrm{T}}A^{\mathrm{T}}A\boldsymbol{x}}{\boldsymbol{x}^{\mathrm{T}}\boldsymbol{x}}=\frac{\boldsymbol{x}^{\mathrm{T}}S\boldsymbol{x}}{\boldsymbol{x}^{\mathrm{T}}\boldsymbol{x}} $$ 如何求解 λ 最大值, 以及取得最大值的时候的 x 向量 ?

首先想到可以求导. 结果如下 : 2Sx = 2λx 这说明 x 是矩阵 S = ATA 的特征向量, 同时 λ 是对应的特征值(也是 A 的奇异值 σ). 因此 λ1 = σ12, x = v1. λn 对应的特征向量 x 则是 极小值点. 而 λ1 和 λn 之间的特征向量则是 R(x) 的驻点.

目标2 : 最大化$\frac{\|A x\|}{\|x\|}$, 其中要求 v1Tx = 0.

最大值是 σ2, x = v2.

除了求导, 还可以将 x 写成 v 的线性组合(由于 dim(V) = n, 任意 n 维向量都可以写成 v′s 的线性组合) : x = c1v1 + ... + cnvn xTx = c12 + ⋯ + cn2

xTSx = λ1c12 + ⋯ + λncn2

因此 Rayleigh 商可以写成 : $$ R(x) = \frac{x^T S x}{x^T x} = \frac{\lambda_1 c_1^2 + \cdots + \lambda_n c_n^2}{c_1^2 + \cdots + c_n^2}. $$

$$ \begin{align*} R(x) &=\frac{\lambda_1\left(c^2_1 + \dots + c^2_n\right) + \left((\lambda_2-\lambda_1)c^2_2+\dots + (\lambda_n-\lambda_1)c^2_n\right)}{c^2_1 + \dots + c^2_n} \\ &= \lambda_1 + \frac{(\lambda_2-\lambda_1)c^2_2+\dots + (\lambda_n-\lambda_1)c^2_n}{c^2_1 + \dots + c^2_n} \end{align*} $$

只要取 λ1 为特征值中的最大值, 之后的项均 ≤0, 从而 R(x) 达到最大值. 此时有 c1 为任意非零值, 其余 c2 = ... = cn = 0.
